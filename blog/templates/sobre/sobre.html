Sou Rafael Monteiro, desenvolvedor web de Campos do Jordão. Atualmente graduando Análise e Desenvolvimento de Sistemas no IFSP.

     

Conhecimento no desenvolvimento web básico, já tendo realizado a estruturação de diversas páginas, bem como a criação de material didático para a disciplina de desenvolvimento web 1 (básico) aplicada no Instituto Federal de São Paulo, campus Campos do Jordão. Clique para ver um exemplo.


Também desenvolvedor de jogos eletrônicos, com colaborações realizadas ao grupo de desenvolvimento de jogos do IFSP-CJO, para o projeto de um jogo que visa simular um ambiente de um restaurante fast-food cooperativo, utilizando a ferramenta Construct2, onde o principal objetivo é o gerenciamento d recursos.


Existem também o trabalho paralelo com o desenvolvimento de jogos utilizando o framework Phaser.js para JavaScript, utilizando esta ferramenta é possível criar jogos de maneira simples para o ambiente web. Clique para ver meus trabalhos.


Por fim participei de um projeto de extensão, que por sua vez tinha o objetivo de concientizar jovens do ensino público local do uso adequado da internet, apresentando as boas práticas, ferramentas para se protegerem e os perigos que pairam sobre a rede.
